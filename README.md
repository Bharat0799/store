📌 Project Overview
LuxeStore is a frontend-only e-commerce application that replicates a real Shopify store experience, including:

Product browsing
Collection filtering
Add to cart functionality
Checkout flow simulation
Clean and modern UI design


🎯 Features
🏠 Homepage

Hero banner with CTA
Featured collections
Best-selling products
Testimonials section
Newsletter signup

🛒 Product & Collection

Product listing page (Shop)
Category/collection filtering
Product detail page with:

Images
Description
Pricing
Variants (if applicable)



🧺 Cart System

Add/remove products
Quantity updates
Dynamic price calculation

💳 Checkout Simulation

Form-based checkout flow
Shipping details
Order summary
Confirmation page

👤 User Authentication (Frontend Only)

Login UI (mock authentication)
Session handling using localStorage


🎨 UI/UX Highlights

Clean and modern design (inspired by Shopify)
Mobile-first responsive layout
Smooth user flow from homepage → checkout
Conversion-focused sections (CTA, trust badges, etc.)


🤖 AI Assistance & Optimization
AI tools like ChatGPT were used for:

Layout structuring (homepage sections)
Copywriting (product descriptions, CTAs)
SEO meta descriptions
UX improvements and navigation planning

All outputs were manually refined for production quality.

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript, Tailwind CSS
State Management: LocalStorage
Deployment: GitHub Pages
